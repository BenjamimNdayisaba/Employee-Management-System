Employee Management System

A full-stack web application for managing employees, tasks, and administrative operations.
Built with React (frontend), Node.js + Express (backend), and MySQL (database).

Features

Employee management: add, update, and organize employee records

Task management: create, assign, and track tasks with priorities and status updates

Admin dashboard for system control and overview

Secure user authentication for employees and administrators

Task tracking with due dates, priority levels, and progress status

MySQL database integration for persistent data storage

Getting Started
Prerequisites

Node.js and npm

MySQL

Git

Installation

Clone the repository

git clone https://github.com/BenjamimNdayisaba/Employee-Management-System.git
cd Employee-Management-System


Backend setup

cd "Server Side"
npm install
npm start


Frontend setup (open a new terminal)

cd "Client Side"
npm install
npm start


Database setup

Create a database named employeems using MySQL or Navicat

Run the SQL scripts located in Server Side/ (including create_tasks_table.sql and related files)

Usage

Start the backend server

Start the React development server

Log in with admin or employee credentials

Use the dashboard to manage employees and tasks
